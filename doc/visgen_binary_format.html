visgen output binary file format.

Shorthand:
TIMBLCK = 8-byte string consisting of "timblck\0"
VISBLCK = 8-byte string consisting of "visblck\0"
main_header = (simname, time, field_center, field_name, stokes, n_time_blocks)
time_block_header = (start_time, duration, integration_time, n_IFs, freq_span*n_IFs)
freq_span = (IF_frequency, bandwidth)
visblock  = (u,v,w,integration_sequence_number,station1, station2, n_freq (REDUNDANT: same as n_IF), visfreq*n_freq)
visfreq   = (freq_index,n_corr_chan, visgroup*n_corr_chan)
visgroup  = (corr_chan,flag,weight,complex_vis*4)
